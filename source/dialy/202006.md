## STM32におけるBOOT PIN
- boot1 : boot0 :: mode
- x : 0 :: Flash program memory
- 0 : 1 :: System memory
- 1 : 1 :: Embedded SRAM

## ESP-WROOM-02におけるBOOT PIN
- io0 : io2 : io15 ::mode
- Low : High : Low :: UART Download Mode (Write Mode)
- High: High : Low :: Flash Boot Mode (Run Mode)
- Low : Low  : High:: SDcard Boot Mode
